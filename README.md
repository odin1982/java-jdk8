# COLLECTIONS
## Diferencia entre Collection y Collections
~~~
La diferencia es que Collection es una interfaz que continene todas las Colecciones y Collections es una utileria.
~~~

## Collection Interface
~~~
La interfaz raiz en la jerarquia de las colecciones. 
Representa un grupo de objetos conocidos como elementos. 
Algunas colecciones permiten duplicados otras no.
Algunas son ordenadas y otras no.
Esta interfaz no proporciona implementaciones directas, se tendria que usar subinterfaces como List o Set
Se utiliza para pasar Colecciones y manipularlas
Muchos metodos en las interfaces de Collections Framework se definen en terminos del metodo equals.Un ejemplo contains(Object o)
~~~
