# ENUM
 - Es una clase que representa una enumeración.
 - No puedes comparar un int con un enum.
 - Se puede crear un enum de un String con valueOf
 - No se puede extender un enum
 - El constructor de un enum DEBE ser privado

##### Declaration
```
public enum Season{
	WINTER, SPRING, SUMMER, FALL;
}
```

##### ENUM.values()
Este metodo obtiene todos los valores del enum y los convierte en un array.
